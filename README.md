# Projecte Solar

Aplicació web per a la gestió d'instal·lacions de plaques solars a l'Horta de València.

## Tecnologies

- **Angular 21**
- **Bootstrap 5**
- **TypeScript**

## Requisits

- Node.js 18 o superior
- Angular CLI (`npm install -g @angular/cli`)

## Instal·lació

```bash
npm install
```

## Ús

```bash
ng serve
```

Obre el navegador a `http://localhost:4200`.
